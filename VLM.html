Visual Language Model (VLM) Project

This project introduces a 500M-parameter Visual Language Model (VLM) designed for real-time perception and multimodal reasoning. The model runs locally on my GPU and is deployed through a fast Flask-based interface, enabling live camera analysis directly from any mobile device.

In its general-purpose mode, the VLM can recognize a wide variety of everyday objects such as tools, fruits, furniture, and common indoor/outdoor items. However, its architecture supports domain-specific fine-tuning, making it suitable for specialized applications in:

  • Precision Agriculture: crop disease detection, weed mapping, fruit maturity evaluation, yield estimation
  • Robotics & Perception: semantic scene understanding, object grounding, navigation assistance, HRI
  • Mechanical Engineering: surface defect detection, quality inspection, component recognition

The system provides not only predictions but also natural-language explanations, helping researchers interpret the model’s reasoning process. This makes the VLM valuable for field robotics, autonomous monitoring, and digital agriculture workflows where edge-based intelligence is essential.
