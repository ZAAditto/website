CNC Machine

I started this project from scratch. First, I designed the machine in Autodesk Fusion 360, which provided the necessary dimensions. I then collaborated with a carpenter to construct the machine's body, using the CAD model to guide the workers. Following that, I handled the electrical and coding aspects. The machine eventually worked satisfactorily and with precision. This project was very challenging for me. The machine can draw any given image by converting a sketch into G-code, which is then inserted into the ‘Open Build’ software. Additionally, a laser can be used instead of a pen to transform the machine into an engraving tool.
