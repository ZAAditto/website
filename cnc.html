CNC Machine

I started this project from scratch. First, I designed the machine in Autodesk Fusion 360, which provided the necessary dimensions. I then collaborated with a carpenter to construct the machine's body, using the CAD model to guide the work. After that, I handled the electrical and coding aspects. The machine eventually worked satisfactorily and with precision. I used Inkscape software to convert the images into G-code, which is then inserted into the 'Open Build' software to draw the design. Additionally, a laser can be used instead of a pen, transforming the machine into an engraving tool.
